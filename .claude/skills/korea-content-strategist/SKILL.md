# korea-content-strategist — 한국형 콘텐츠 전략 진입점

## 무엇을 하는가

사용자가 **사업/제품 + 타깃 + 마케팅 목표**를 주면, 오케스트레이터가
**단일 워크플로우로 직접 설계**(오리지널 Skill 토폴로지 — subagent
없음)해 ① 타깃·인사이트 ② 메시지 기둥 3~5개 ③ 톤·보이스 ④ 채널별
앵글 ⑤ 콘텐츠 아이디어 ⑥ 실행 캘린더 골격을 정리하고, 콘텐츠 전략
마크다운 1개를
`./specs/claude-for-x-kr/korea-content-strategist/<slug>/<today>/`에
저장한다.

이것은 Anthropic Small Business의 **Content Strategist** Skill을
기능·토폴로지·출력 불변으로 한국에 이식한 것이다. 오리지널 명세가
단촐(커넥터·subagent 없음)하며, 그 단촐함을 우리 패턴으로 부풀리지
않는다. 변형은 사실상 **문화 레이어 하나**(美 마케팅 카피·채널 관행
→ 한국 콘텐츠·카피·채널 생태계: 네이버 블로그·카카오·인스타·숏폼·
당근 등, 한국 소비자 정서)뿐이다.

## 언제 트리거되는가

- "콘텐츠/마케팅 메시지 전략 짜줘·기획해줘"
- "이 사업 어떻게 알릴지", "채널별 콘텐츠 방향"
- "메시지 기둥 정리", "SNS 콘텐츠 전략"

## 트리거되지 않는 경우

- 계약서 검토 → `/korea-contract-reviewer`
- 상장사 실적·섹터 리서치 → korea-earnings-reviewer / market-researcher
- 일반 웹 조사 → 일반 WebSearch
- 코드베이스 분석 → `/reverse`

## 어떻게 동작하는가

이 스킬이 트리거되면 **`/korea-content-strategist` 커맨드의 실행
순서를 그대로 수행**한다:

1. **Step 0** — 입력(사업·타깃·목표) 수령(비면 한 번만 요청),
   정규화, 저장 경로 결정.
2. **Step 1** — 오케스트레이터가 단일 워크플로우로 직접 전략 설계
   (타깃 → 메시지 기둥 → 톤 → 채널별 앵글 → 아이디어 → 캘린더).
   subagent 없음(오리지널 Skill 토폴로지 보존). 규제 분야는 표현
   규제 확인 필요 명시.
3. **Step 2** — 마크다운 1개 저장 → grep 결정적 자가검증(메시지
   기둥 섹션 존재 포함) → 로그 append → 요약 보고.

입력이 비면 한 번만 묻고, 그 외엔 끝까지. 상세 계약은
`.claude/commands/korea-content-strategist.md`를 따른다. 메시지
전략 설계가 본분이며 콘텐츠 대량 생성·광고 집행 대행은 하지 않는다.
